Detailed Description

Can be customized to add new error codes. All MONARCH_API_ error codes will be piped with SIGFOX_MONARCH_API_xxx return code.

IMPORTANT : SFX_ERR_NONE return code is mandatory when no error for each MCU_API_xxx MONARCH_API_xxx RF_API_xxx REPEATER_API_xxx or SE_API_xxx functions.
